# trianglify

htmlwidget for [trianglify](http://qrohlf.com/trianglify/).

## Installation

``` r
# install.packages("devtools")
devtools::install_github("JohnCoene/trianglify")
```

## Example

See the [website](https://trianglify.john-coene.com) for more.

``` r
library(trianglify)

trianglify()
```
